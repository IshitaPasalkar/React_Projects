## 📝 Project Title: **Feedback App**

### 📌 Description:

The **Feedback App** allows users to submit their name and a feedback message. Once submitted, the app displays the submitted information on the screen, giving users a visual confirmation of their input.

### 🎯 Learning Objectives:

By completing this project, students will:

✅ Learn to use the `useState` hook to manage multiple pieces of form state
✅ Understand how to create controlled components using `value` and `onChange`
✅ Handle form submissions gracefully
✅ Conditionally render content after submission
✅ Clear input fields after data is submitted
