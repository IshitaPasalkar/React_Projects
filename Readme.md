## ▶️ Getting Started

### Clone the repository

```bash
git clone https://github.com/IshitaPasalkar/React_Projects.git
```

### Navigate to the project folder

```bash
cd React_Projects
```

### Install dependencies

```bash
npm install
```

### Start the development server

```bash
npm run dev
```

> Open the specific project folder you want to work on before installing dependencies and running the development server.
