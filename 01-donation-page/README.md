## 🎯 Project Title: **JSX Playground – Personal Profile Page**

This project will help you practice:

- Writing **JSX syntax**
- Using **JavaScript expressions** inside JSX
- Understanding how **HTML-like code** mixes with JS

---

## 🧱 What We’ll Build

A simple profile page that displays:

- Your name
- Your favorite programming language
- A short bio
- A profile picture
- A list of hobbies

✅ All built using JSX — no props, no state, no interactivity.
